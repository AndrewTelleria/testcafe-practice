# Test Cafe Practice

To install Test Cafe:

```npm install -g testcafe```

Run locally:

```testcafe chrome tests```

Install Gulp:

```npm install -g gulp```
```npm install --save-dev gulp```

Install plugin that integrates Test Cafe and Gulp:

```npm install --save-dev gulp-testcafe```

Run the task:

```gulp run-testcafe-tests```
